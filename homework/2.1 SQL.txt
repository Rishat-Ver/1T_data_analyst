Задания:

1. Напишите запрос, который вернёт колонки “Общий доход”, “Возраст” , “ID” именно в таком порядке.
SELECT "Annual_Income" as "Общий доход",
        "Age" as "Возраст",
        "CustomerID" as "ID"
    FROM mall_customers;

------------------------------------------------------------------------------------------------------
2. Напишите запрос, который вернет всех посетителей магазина возрастом 50 лет.
SELECT *
  FROM mall_customers
 WHERE "Age" = 50;

------------------------------------------------------------------------------------------------------
3. Напишите запрос, который вернет всех посетителей магазина с общим доходом более 70 или женщин.
SELECT *
  FROM mall_customers
 WHERE "Annual_Income" > 70;

------------------------------------------------------------------------------------------------------
4. Напишите запрос, который вернет всех ID c 20 по 51
SELECT "CustomerID" as "ID"
  FROM mall_customers
 WHERE  "CustomerID" >= 20 and "CustomerID" <= 51;

------------------------------------------------------------------------------------------------------
5. Напишите запрос, который вернет всех покупателей чей Общий скор больше общего дохода, но меньше ID.
SELECT *
  FROM mall_customers
 WHERE  "Spending_Score" > "Annual_Income" and "Spending_Score" < "CustomerID";
